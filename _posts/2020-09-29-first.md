## 1. Github repository 만들기
<br>

- github repository - 이름은 아이디.github.io로 만들어야한다
그렇지 않으면 오류가 날수도..!

- 깃허브 레파지토리 주소를 복사한 후 폴터 하나를 만들어 clone해준다

```
git clone [레파지토리 주소]
```
